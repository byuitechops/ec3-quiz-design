Hypothetical Scenerios

GOAL: Understand hypothetical situations

Learn

“Hypothetical” describes something that is not real, but is possible. For example, it is possible to eat a taco tomorrow. It is not sure that you will eat a taco tomorrow. But it is possible. This is a hypothetical situation. We use conditional verb tenses for hypothetical situations.


Some hypothetical situations are in the past. They did not happen in the past, but they were possible. For example, it was possible for you to eat a taco yesterday. You did not eat a taco. But it was possible. This is hypothetical because it is not real. It is also in the past. For hypothetical situations that were possible in the past, but did not happen, use past perfect and present perfect tenses, with a modal (would or could):


Hypothetical, unreal situation in the past:


If + subject + had + verb-ed, then + subject + modal + have + verb-ed.

If he had studied, then he would have learned.

Vocabulary

Hypothetical: something that is not real, but is possible.

Situation: a set of things that are happening or conditions that exist.

Conditional: something must be true for the second thing to be true.

Study

Hypothetical situations in the past:

If I had exercised, I could have had better health.

If you had practiced piano, you would have learned to play.

If he had studied more, he would not have failed the test.

Practice

Talk to a friend. Tell your friend about hypothetical situations in the past.

Possible answers:

If I had moved to the U.S., I could have learned English.

If I had studied harder, I would have learned more.

If I had talked to him, I would have understood better.
