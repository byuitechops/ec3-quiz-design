COMPLEX QUESTIONS

GOAL: Learn to make Indirect Questions

Learn

English has many ways to ask questions. Some are simple and some are complicated.


One complicated question type is indirect questions. We use indirect questions to be polite or formal. For example, if I am at an expensive restaurant, and I need to use the bathroom, I use an indirect question like “Could you please tell me where the bathroom is?”


Form indirect questions by using an indirect question phrase, and then adding the rest of the question after the phrase.


Indirect question phrases:

  • Could you (please) tell me...
  • Do you know...
  • I was wondering...
  • Do you have any idea...
  • I'd like to know...
  • Would it be possible...
  • Is there any chance...

Vocabulary

Polite: respectful actions and manners.

Formal: following rules carefully and being very polite.

Study

  • Could you tell me when the party starts?
  • Do you know what time it is?
  • I was wondering if you could help me.
  • Do you have any idea how to do the homework?
  • I’d like to know when I should arrive.
  • Would it be possible to get a ride?
  • Is there any chance you can help me?

Practice

Talk with a friend. Ask your friend indirect questions about his/her life.

Possible Answers

  • I’d like to know why you’re studying English.
  • Is there a chance you can watch my kids tonight?
  • Do you know when we’ll finish this class?
