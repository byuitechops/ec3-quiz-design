Hypothetical Scenarios

GOAL: Understand hypothetical situations

Learn

“Hypothetical” describes something that is not real, but is possible. For example, it is possible to eat a taco tomorrow. It is not sure that you will eat a taco tomorrow. But it is possible. This is a hypothetical situation. We use conditional verb tenses for hypothetical situations.


Some hypothetical situations are usually true when they happen. For example, if you don’t brush your teeth, you get cavities. This is hypothetical because it is not happening right now, so it is not real. But it is usually true when it happens. For hypothetical situations that are usually true, use simple present tense sentences.


Hypothetical, usually true.

If + subject + verb + Object, then + subject + verb + object.

If you eat junk food, then you will get fat.

If he works hard, then he succeeds.

If you love God, then you serve him.

Vocabulary

Hypothetical: something that is not real, but is possible.

Situation: a set of things that are happening or conditions that exist.

Conditional: something must be true for the second thing to be true.

Study

Hypothetical situations that are usually true:

If you study hard, you learn.

If we serve others, we are happy.

If she practices piano, she improves her piano playing.

Practice

Talk to a friend. Tell your friend about hypothetical situations that are usually true in your life.

Possible answers

If I get up early, I feel good.

If I work all day, I am tired.

If I keep the commandments, I am happy.
