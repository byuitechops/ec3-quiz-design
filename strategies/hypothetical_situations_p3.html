HPOTHETICAL SCENARIOS

GOAL: Understand hypothetical situations

Learn

“Hypothetical” describes something that is not real, but is possible. For example, it is possible to eat a taco tomorrow. It is not sure that you will eat a taco tomorrow. But it is possible. This is a hypothetical situation. We use conditional verb tenses for hypothetical situations.


Some hypothetical situations are unlikely or unrealistic. It is possible for them to be true, but they probably never will be. For example, if you went to the moon, you would see the stars. This is hypothetical because it is not real. It is possible, but it probably will never be true. It is very unlikely and unrealistic. For hypothetical situations that are unlikely or unrealistic, use simple past tense and a modal (would or could):


Hypothetical, unlikely or unrealistic:


If + subject + verb-ed + Object, then + subject + modal + verb + object.

If you walked 100 miles, then you would be tired.

Vocabulary

Hypothetical: something that is not real, but is possible.

Situation: an event or location.

Conditional: something must be true for the second thing to be true.

Unlikely: probably will not happen. Almost sure that it will not happen.

Unrealistic: not real, not true. Probably wil not happen.

Study

Hypothetical situations that are unlikely or unrealistic:

If I had a billion dollars, I would buy cars for all of my friends.

If you danced for a month, you would feel sore.

If they talked to the president of the U.S., they could ask for help.

Practice

Talk to a friend. Tell your friend about hypothetical situations that are unlikely or unrealistic.

Possible Answers

If I were rich, I would give you some money.

If I had twenty children, I could not feed them all.

If I worked for the king, I could talk to him.
